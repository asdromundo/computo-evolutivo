Utilizado 

SO : ubuntu 22.04.04 
Python 3.10.9
Bibliotecas utilizadas : 
        - numpy
        - matplotlib
        - scipy 

Instrucciones para NUMPY: 
    
    pip install numpy

Instrucciones para MATPLOTLIB 
Se intstala tanto con pip como con miniconda : 
    
    pip install matplotlib

    รณ 

    conda install -c conda-forge matplotlib

Ejecucion del programa : 

Estando dentro del directorio de la tarea '316032708', el script en donde se ejecutan los algoritmos es src/executable.py y el comando es 

    python3 src/executable  

Esto producira una ejecucion individual de dimension 20, para ejecuciones promedio es necesario comentar la linea 112y descomentar linea 114 de 
src/executable.
